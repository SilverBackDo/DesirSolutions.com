# Statement of Work Template

This template is the trust-package summary version of the Desir Solutions SOW.

Each SOW should define:

- client legal name
- project name
- business objective
- in-scope services
- out-of-scope items
- deliverables
- timeline and review dates
- assumptions and dependencies
- fees and payment schedule
- acceptance process
- special security or data-handling terms

## Default Working Rules

- no fee-bearing work begins without signature
- deliverables are reviewed within `5 business days` unless otherwise stated
- material changes require written change approval
- payment terms are `Net 15` unless otherwise stated
- security or privacy addenda are attached only when required by the engagement

Use the client-facing contract in the private operating repository for the full signature-ready document.
